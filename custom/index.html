Custom CSS with Bootstrap

This demonstrates building a custom CSS file with Bootstrap's source Sass files, managed via npm. It's not a complete solution, and skips compiling Bootstrap's JavaScript.

This demo includes:

  • index.html — the page you're looking at.
  • custom.scss — a customized version of bootstrap.scss where we normally import every Sass file.

The normal import stack is commented out, save for a handful of components:

  • functions, variables, and mixins — useful tools for interacting with and customizing Bootstrap's default styles.
  • reboot — our opinionated browser normalization styles.
  • type — our default typography styles.
  • utilities — all our utility classes for quick prototyping.

You can compile this CSS with npm run custom.
